


# 🎱 Magic Ball 8 Ultimate

**Интеллектуальный PWA-помощник с гибридной логикой ответов и 100% приватностью данных.**

Этот проект — не просто цифровая игрушка, а полноценное интерактивное веб-приложение, которое сочетает в себе магию классического "Магического шара 8" с современными веб-технологиями. Приложение работает **полностью офлайн**, не отправляет личные вопросы на сторонние серверы и использует продвинутые локальные алгоритмы для анализа запросов.

Часть бесплатной социальной экосистемы **Anti-AI Shield**.

## ✨ Ключевые особенности

*   🔒 **100% Приватность и Офлайн-режим:** Вся базовая логика, история и кэширование работают локально в браузере (LocalStorage, Service Worker). Ваши личные вопросы не покидают ваше устройство.
*   🧠 **Умная локальная логика:** 
    *   Весовая система анализа ключевых слов (стемминг).
    *   Безопасный математический парсер (поддерживает цепочки вычислений без использования опасного `eval`).
    *   Контекстные и тематические ответы (любовь, карьера, здоровье).
*   🌐 **Гибридный онлайн-режим:** При наличии интернета приложение может обратиться к API Википедии для поиска фактов, используя умный фолбэк (поиск по заголовкам при ошибке 404) и фильтрацию страниц-неоднозначностей (disambiguation).
*   🎙️ **Голосовое управление:** Полная поддержка Web Speech API для голосового ввода вопросов и озвучивания ответов (с обработкой ошибок и доступностью).
*   🎨 **Премиум UI/UX:** 
    *   Стеклянный 3D-дизайн шара с анимацией встряхивания и эффектом частиц (Canvas API).
    *   Адаптивный размер шрифта с защитой от переполнения.
    *   Переключение темной и светлой темы.
    *   Тактильная отдача (Vibration API) для мобильных устройств.
*   💾 **Управление данными:** Экспорт и импорт истории диалогов в формате JSON.

## 🛠️ Технологический стек

*   **HTML5 / CSS3** (Glassmorphism, CSS-анимации, адаптивная верстка, переменные тем).
*   **Vanilla JavaScript (ES6+)** (Модульная архитектура, Async/Await, Web Audio API для генерации звуков).
*   **PWA** (Service Worker, Manifest.json для установки на главный экран).
*   **Web APIs:** SpeechRecognition, SpeechSynthesis, Vibration, LocalStorage, Canvas.

## 🚀 Как запустить

1. Клонируйте репозиторий или скачайте архив с файлами.
2. Убедитесь, что в папке находятся: `index.html`, `game.html`, `sw.js`, `manifest.json` и `favicon.ico`.
3. Откройте `index.html` или `game.html` в любом современном браузере.
4. *(Опционально)* Для полноценной работы PWA и Service Worker рекомендуется запускать через локальный сервер (например, расширение "Live Server" в VS Code или `python -m http.server`).
